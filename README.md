# movie-test

This Is API for Movie List

## Install

    npm install

## Set .env file

    PORT=

    BASE_URL=
    PATH_FILE=
    NODE_ENV =

    DB_USER=
    DB_PASS=
    DB_NAME=
    DB_HOST=
    DB_PORT=
    DB_DIALECT=

## Run the app

    npm start

## Run the tests

    npm run test

# REST API

The REST API to the Movie List is described below.

## Get list of Movies

### Request

`GET /Movies`

## Create a new Movie

### Request

`POST /Movies`

## Get a Movie Detail

### Request

`GET /Movies/:id`

## Update a Movie

### Request

`PATCH /Movies/:id`

## Delete a Movie

### Request

`DELETE /Movies/:id`
